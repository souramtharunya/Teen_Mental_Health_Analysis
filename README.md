# Teen Mental Health Data Analysis

## 📊 Project Overview

This project analyzes teen mental health data to identify patterns in stress, anxiety, and lifestyle factors.

## 🛠️ Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

## 📈 Features

* Data Cleaning
* Data Visualization
* Correlation Analysis
* Insight Generation

## 📁 Files

* analysis.ipynb → Jupyter Notebook with code
* Teen_Mental_Health_Dataset.csv → Dataset
