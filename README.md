# 🧠 Teen Mental Health Data Analysis

## 📊 Project Overview

This project analyzes teen mental health data to identify patterns in stress, anxiety, and lifestyle habits such as screen time, sleep, and social media usage.

## 🎯 Objectives

* Understand key factors affecting teen mental health
* Identify relationships between lifestyle and stress levels
* Generate insights using data visualization

## 🛠️ Tools & Technologies

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

## 📁 Dataset

The dataset contains information about:

* Stress levels
* Anxiety levels
* Sleep duration
* Social media usage
* Other lifestyle factors

## 📈 Analysis Performed

* Data Cleaning (handling missing values)
* Exploratory Data Analysis (EDA)
* Correlation Analysis
* Visualization of trends and distributions

## 🔍 Key Insights

* Higher screen time is associated with increased stress levels
* Sleep duration shows a negative correlation with anxiety
* Lifestyle habits significantly impact mental health

## ▶️ How to Run

1. Download the repository
2. Open `analysis.ipynb` in Jupyter Notebook
3. Run all cells step-by-step

## 📌 Files

* `analysis.ipynb` → Complete analysis code
* `Teen_Mental_Health_Dataset.csv` → Dataset
